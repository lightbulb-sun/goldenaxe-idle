# Golden Axe (Genesis) Idle Timer Patch

## Changes
Disables idle timers for demo in title screen and main menu
for version v1.1 of the Genesis game
*Golden Axe*
in accordance with this
[reddit thread](https://www.reddit.com/r/romhacking/comments/1jtbykx/golden_axegenesis_title_screen_idle_timer_hack/)
by user
[Morpheus414](https://www.reddit.com/user/Morpheus414).

## Download
The latest release can be found on the
[releases page](https://github.com/lightbulb-sun/goldenaxe-idle/releases).
Patch the original ROM with one of the `.bsdiff`, `.ips` or `.bps` files
to create the hack.
